Helper recursive function that checks if the tree is balanced.

Parameters
head_nodeRaw pointer to a Node, which will be the head Node. Since it's a unique pointer, we need to call it with head.get(). Recall that a tree is balanced if, for each Node, the height of the left and right subtree is at most 1 and each subtree of the tree is balanced.
