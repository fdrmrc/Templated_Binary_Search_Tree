Templated Binary Search Tree
Template Binary Search Tree

A C++ implementation of a Templated Binary Search Tree using smart pointers, tested with the GoogleTest framework. You can find a Doxygen documentation at: https://fdrmrc.github.io/Templated_Binary_Search_Tree/html/index.html

Authors:

  • Marco Feder

  • How to compile

First, clone the repo to your local machine.

Using Makefile:

  • open a terminal
  • move to the project folder
  • type make to build the executable exe
  • ./exe.x from command line

From command line:

  • $ g++ -o exe.x -Wall -Wextra -g -std=c++14 -I include main.cpp
  • then do ./exe.x

To check memory leaks with Valgrind and see the heap/leak summary:

  • valgrind ./exe.x --leaks-check=full

The current version doesn't leak memory, as can be verified.

Unit testing

To check the goodness of our functions, we used the testing framework Google Test. In particular, we provide tests for the each one of the implemented classes. If you have Google Test installed on your machine, you can just use the Makefile provided in the unit_tests folder, or alternatively:

  • g++ test_all.cpp -lgtest -lgtest_main -pthread -o test_all -std=c++14
  • ./test_all
